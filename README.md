Simply require all available build tasks.
=========================================

This is useful to only have to put one dependency into `require-dev` when utilizing all phpcq tasks.

Usage
-----

Add to your `composer.json` in the `require-dev` section:

```
"phpcq/all-tasks": "~1.0"
```
